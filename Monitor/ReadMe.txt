И для сервера и для клиента используется коспиллятор g++ из пакета GNU(для Windows).

Для начала работы программы нужно
1.Установить сервер(setup.bat в папке Server)
	а)После запуска сервера откроется консольное окно, куда будут выводится данные о поделючённых клиентах.
	б)также появится окно, в поле ввода которое нужно ввести номер подключённого пользователя чтобы получить снимок его экрана.
2.Установить клиент(setup.bat в папке Client)
3.Либо запустить клиент вручную, либо перезагрузить ПК, чтобы программа запустилась автоматически.
