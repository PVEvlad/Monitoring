И для сервера и для клиента используется коспиллятор g++ из пакета GNU(для Windows).

Установка Сервера
1.В командной строке перейдите в директорию Сервера.
2.Для компилляции введите следующее:
	g++ server.cpp -lWS2_32 -lIPHlpApi -lnetapi32 -lGdi32 -o server.exe
3.Сервер может быть запущен двойным кликом левой кнопки мыши по .exe файлу.

Установка клиента
1.В командной строке перейдите в директорию Клиента.
2.Для компилляции введите следующее:
	g++ client.cpp client_m.cpp -lWS2_32 -lIPHlpApi -lnetapi32 -lgdi32 -o client.exe
3.Для помещения программы в список на автозапуск потребуется следующая команда:
	copy client.exe "%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup"
4.Далее для запуска программы можно либо дважды щёлкнуть левой кнопкой мыши по исполняемому файлу, или перезапустите ПК.
